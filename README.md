# dotfiles-ssh-work

Work SSH config. Cloned directly into `~/.ssh` by `install.sh --env work`.

See the main [dotfiles](https://github.com/charlief0x/dotfiles) repo for setup instructions.

## Contents

- `config` — global SSH defaults
- `config.d/github` — `github-work` alias (Fetch GitHub account via 1Password SSH agent)
- `config.d/workbit` — workbit host
- `2024-09-09_fetch_github_prod.pub` — work GitHub public key

## Keys

SSH keys are stored in 1Password. See the personal [dotfiles-ssh](https://github.com/charlief0x/dotfiles-ssh) README for full setup instructions.
